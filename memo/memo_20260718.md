### スモークテスト（＝Step 1）

```bash
PYTHONPATH=src python -c \
'from factual_scene_graph.parser.scene_graph_parser import SceneGraphParser; p=SceneGraphParser("lizhuang144/flan-t5-base-VG-factual-sg", device="cpu"); print(p.parse(["2 beautiful pigs are flying on the sky with 2 bags on their backs"], beam_size=1, return_text=True)[0])'
```

#### 解説：このコマンドが行っていること

- `PYTHONPATH=src`：この実行に限り、Python の import 検索パスに `src/` を追加する。FACTUAL は pip インストールしていないため、これで `src/factual_scene_graph/` を import できるようにする（環境変数を付けたコマンドの前置き）。
- `python -c '...'`：`'...'` の中の Python コードをその場で実行する（ファイルを作らずに1行で試す形。`\` は行継続で、次行の `'...'` につないでいる）。
- `from factual_scene_graph.parser.scene_graph_parser import SceneGraphParser`：解析器クラス `SceneGraphParser` を読み込む。
- `p = SceneGraphParser("lizhuang144/flan-t5-base-VG-factual-sg", device="cpu")`：解析器を初期化する。指定した学習済みモデル（Flan-T5 を Visual Genome で微調整したもの）の**トークナイザとモデルを Hugging Face から取得**（初回はダウンロード、以後キャッシュ）し、**CPU 上に載せる**。`device="mps"` にすると Apple Silicon GPU を使う。
- `p.parse([...], beam_size=1, return_text=True)`：実際に解析する。
  - 第1引数は**文のリスト**（複数文をまとめて渡せる。ここでは1文）。
  - `beam_size=1`：デコードを**貪欲法**で行う（ビームサーチの幅=1）。大きくすると探索が広がり出力が変わりうる。
  - `return_text=True`：結果を**文字列**（`( a , b , c ) , ...` 形式）で返す。`False` にすると `entities`／`relations` の構造化オブジェクトになる。
  - 戻り値は**入力ごとのリスト**なので、`[0]` で最初の文の結果を取り出す。
- `print(...)`：取り出した三つ組の文字列を表示する。

要するに「**srcを読めるようにして → 学習済みモデルを読み込み → 1つの英文をCPUで貪欲デコードして Scene Graph（三つ組文字列）に変換し → 表示する**」という一連の処理。

### 最小構成 と フル構成 の違い

- **最小構成（上記・現在の `venv`）**：parser の**推論だけ**動かす用。torch / transformers==4.46.3 / nltk / tqdm / tabulate / sentencepiece。
- **フル構成（`requirements.txt`）**：CLIP・sentence\_transformers==2.2.2・spacy・scipy・discosg 等を含む。**SPICE / SoftSPICE 評価や DiscoSG-Refiner** を使う場合のみ必要（重い・インストールでつまずきやすい）。今回の理解フェーズでは不要。

Python 3.11.6 ／ torch 2.12.1 ／ transformers 4.46.3 ／ nltk 3.9.4 ／ tqdm 4.68.3 ／ tabulate 0.10.0 ／ sentencepiece 0.2.1 ／ numpy 2.4.6

### 実行の基本形

- 場所：`FactualSceneGraph/` で `PYTHONPATH=src` を付けて実行。
- モデル：`lizhuang144/flan-t5-base-VG-factual-sg`（通常）／`lizhuang144/flan-t5-base-VG-factual-sg-id`（ID付き）。
- FACTUAL がやること：英語の文 → 三つ組 `( 主体 , 関係/属性 , 対象/値 )` の集合（=Scene Graph）。`is` は属性、それ以外は関係。
